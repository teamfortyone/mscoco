## Dataset
Flickr8k data

### Dataset setup

1. Download the dataset
2. Place it in /data
3. The data directory should look like this:

    ```
    .    
    ├── data
    │   ├── Flickr8k_Dataset
    │   └── Flickr8k_text
    ```
